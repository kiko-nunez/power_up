# Power Up: EV Charging
## The purpose of this app is to allow users to find the nearest EV charging stations, store vehicle information, and leave reviews.

* Python
* Django
* PostgreSQL
* Heroku Deployment
* Materialize Framework
* Trello
* ERD
* GitHub
